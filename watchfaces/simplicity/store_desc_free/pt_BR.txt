Mostrador rico em dados com 12 espaços para todos os seus dados, exibido de forma simples. Totalmente personalizável com gráficos, barras de progresso, 3 fontes, etc.

⭐️ Gratuito vs Pro ⭐️
Este mostrador está disponível sem custo com todas as funções básicas. Recursos premium (marcados com [*]) requerem uma Chave Pro para desbloquear. Você pode comprar uma Chave Pro aqui: https://ko-fi.com/s/e33037b90e Ou comprar a versão PRO aqui: https://apps.garmin.com/apps/4fc31fcb-e422-414b-ac09-660e21ea6dff

⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc) ⭐️
Este mostrador é totalmente otimizado para MIPS, mas as cores parecerão diferentes das imagens, pois estas foram tiradas em um dispositivo Amoled.

⭐️ Configuração ⭐️
Nota: Há dois lugares diferentes com configurações:
* App Garmin IQ (no seu telefone): Configure campos de dados, cores e configurações gerais.
* Configurações no relógio: Acesse opções mais detalhadas como formato de hora, metas e unidades diretamente no relógio segurando o botão menu, selecionando mostrador e tocando personalizar.
