Uma fusão de estética retro clássica e tecnologia de smartwatch contemporânea. Este mostrador combina tipografia nostálgica inspirada em LCD e elementos digitais vintage com visualização de dados moderna, opções avançadas de personalização e princípios de design elegantes. Experimente a harmonia perfeita entre o charme à moda antiga e funcionalidade de vanguarda.
    
⭐️ Grátis vs Pro ⭐️
Este mostrador está disponível gratuitamente com todas as funções básicas. Funcionalidades premium (marcadas com [*]) requerem uma Chave Pro para desbloquear. Pode comprar uma Chave Pro aqui: https://ko-fi.com/s/2dffa075ce Ou comprar a versão PRO aqui: https://apps.garmin.com/apps/c2cf91b0-3c2f-4c05-bdd7-5198f2e588ff

⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Este mostrador está completamente optimizado para MIPS, mas as cores parecerão diferentes como na maioria das imagens, pois estas foram tiradas num dispositivo Amoled. Este mostrador tem muitos temas optimizados para MIPS e suporta gradientes MIPS. Pode encontrar todos os temas incorporados aqui: https://watchface.io/legacy/themes#MIPS

⭐️ Configuração ⭐️
Nota: Há dois locais diferentes com configurações:
* App Garmin IQ (no seu telemóvel vá ao seu dispositivo, mostradores, configurações): Configure campos de dados, cores e configurações gerais.
* Configurações no Relógio: Aceda a opções mais detalhadas como formato de hora, objetivos e unidades diretamente no relógio mantendo o botão menu premido, selecionando mostrador e tocando personalizar.

⭐️ Campos de Dados ⭐️
Todos os campos de dados que a Garmin atualmente fornece a programadores terceiros estão disponíveis, mais extras como suporte ao Glucose Monitor (CGM), Open Weather Map (OWM) e suporte para outras aplicações terceiras.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields . Alguns campos de dados (como prontidão para treino ou HRV) atualmente não estão disponíveis para programadores terceiros, veja aqui: https://watchface.io/faq/can-you-add-training-readiness
