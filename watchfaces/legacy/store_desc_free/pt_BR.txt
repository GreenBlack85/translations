Uma fusão de estética retro clássica e tecnologia de smartwatch contemporânea. Este mostrador combina tipografia nostálgica inspirada em LCD e elementos digitais vintage com visualização de dados moderna, opções avançadas de personalização e princípios de design elegantes. Experimente a harmonia perfeita entre o charme old-school e funcionalidade de ponta.
    
⭐️ Grátis vs Pro ⭐️
Este mostrador está disponível gratuitamente com todas as funções básicas. Recursos premium (marcados com [*]) requerem uma Chave Pro para desbloquear. Você pode comprar uma Chave Pro aqui: https://ko-fi.com/s/2dffa075ce Ou comprar a versão PRO aqui: https://apps.garmin.com/apps/c2cf91b0-3c2f-4c05-bdd7-5198f2e588ff

⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Este mostrador é totalmente otimizado para MIPS, mas as cores parecerão diferentes como na maioria das imagens, pois estas foram tiradas em um dispositivo Amoled. Este mostrador tem muitos temas otimizados para MIPS e suporta gradientes MIPS. Você pode encontrar todos os temas incorporados aqui: https://watchface.io/legacy/themes#MIPS

⭐️ Configuração ⭐️
Nota: Há dois lugares diferentes com configurações:
* App Garmin IQ (no seu telefone vá para seu dispositivo, mostradores, configurações): Configure campos de dados, cores e configurações gerais.
* Configurações no Relógio: Acesse opções mais detalhadas como formato de hora, metas e unidades diretamente no relógio mantendo o botão menu pressionado, selecionando mostrador e tocando personalizar.

⭐️ Campos de Dados ⭐️
Todos os campos de dados que a Garmin atualmente fornece para desenvolvedores terceirizados estão disponíveis, mais extras como suporte ao Glucose Monitor (CGM), Open Weather Map (OWM) e suporte para outros aplicativos terceirizados.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields . Alguns campos de dados (como prontidão para treinamento ou HRV) atualmente não estão disponíveis para desenvolvedores terceirizados, veja aqui: https://watchface.io/faq/can-you-add-training-readiness
