Une fusion d'esthétique rétro classique et de technologie de montre connectée contemporaine. Ce cadran de montre mélange la typographie nostalgique inspirée des écrans LCD et les éléments numériques vintage avec la visualisation de données moderne, les options de personnalisation avancées et les principes de design élégants. Découvrez l'harmonie parfaite entre le charme old-school et la fonctionnalité de pointe.
    
⭐️ Gratuit vs Pro ⭐️
Ce cadran de montre est disponible gratuitement avec toutes les fonctions de base. Les fonctionnalités premium (marquées [*]) nécessitent une Clé Pro pour être débloquées. Vous pouvez acheter une Clé Pro ici : https://ko-fi.com/s/2dffa075ce Ou acheter la version PRO ici : https://apps.garmin.com/apps/c2cf91b0-3c2f-4c05-bdd7-5198f2e588ff

⭐️ Appareils MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Ce cadran de montre est entièrement optimisé pour MIPS, mais les couleurs paraîtront différentes comme dans la plupart des images, car celles-ci sont prises sur un appareil Amoled. Ce cadran de montre a de nombreux thèmes optimisés pour MIPS et supporte les dégradés MIPS. Vous pouvez trouver tous les thèmes intégrés ici : https://watchface.io/legacy/themes#MIPS

⭐️ Configuration ⭐️
Note : Il y a deux endroits différents avec les paramètres :
* App Garmin IQ (sur votre téléphone, allez à votre appareil, cadrans de montre, paramètres) : Configurez les champs de données, les couleurs et les paramètres généraux.
* Paramètres sur la Montre : Accédez à des options plus détaillées comme le format de l'heure, les objectifs et les unités directement sur la montre en maintenant le bouton menu, en sélectionnant cadran de montre et en tapant personnaliser.

⭐️ Champs de Données ⭐️
Tous les champs de données que Garmin fournit actuellement aux développeurs tiers sont disponibles, plus des extras comme le support Glucose Monitor (CGM), Open Weather Map (OWM) et le support pour d'autres applications tierces.
Tous les champs de données disponibles sont listés ici : https://watchface.io/docs/datafields . Certains champs de données (comme la préparation à l'entraînement ou VRC) ne sont actuellement pas disponibles aux développeurs tiers, voir ici : https://watchface.io/faq/can-you-add-training-readiness
