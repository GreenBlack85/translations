Uma fusão de estética retrô clássica e tecnologia contemporânea de smartwatch. Este mostrador combina tipografia nostálgica inspirada em LCD e elementos digitais vintage com visualização de dados moderna, opções avançadas de personalização e princípios de design elegantes. Experimente a harmonia perfeita entre o charme old-school e funcionalidade de ponta.
    
⭐️ Experimente antes de comprar! ⭐️
Antes de comprar este mostrador, confira primeiro a versão gratuita para ter certeza de que é o que você quer e que gosta da aparência no seu relógio: https://apps.garmin.com/apps/f85f4e60-456d-453e-9b35-6cfbe5d33b61

⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Este mostrador é totalmente otimizado para MIPS, mas as cores parecerão diferentes como na maioria das imagens, pois estas foram tiradas em um dispositivo Amoled. Este mostrador tem muitos temas otimizados para MIPS e suporta gradientes MIPS. Você pode encontrar todos os temas integrados aqui: https://watchface.io/legacy/themes#MIPS

⭐️ Configuração ⭐️
Nota: Existem dois lugares diferentes com configurações:
* App Garmin IQ (no seu telefone vá para seu dispositivo, mostradores, configurações): Configure campos de dados, cores e configurações gerais.
* Configurações no Relógio: Acesse opções mais detalhadas como formato de hora, metas e unidades diretamente no relógio segurando o botão menu, selecionando mostrador e tocando personalizar.

⭐️ Campos de Dados ⭐️
Todos os campos de dados que a Garmin atualmente fornece para desenvolvedores terceiros estão disponíveis, mais extras como suporte ao Glucose Monitor (CGM), Open Weather Map (OWM) e suporte para outros apps terceiros.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields . Alguns campos de dados (como prontidão para treino ou HRV) atualmente não estão disponíveis para desenvolvedores terceiros, veja aqui: https://watchface.io/faq/can-you-add-training-readiness
