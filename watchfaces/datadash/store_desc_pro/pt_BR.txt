Mostrador rico em dados com campos personalizáveis, gráficos, diagramas e barras de progresso. Apresenta múltiplos estilos de exibição de dados, atalhos por toque, fontes personalizadas, suporte AOD e extensas opções de personalização de cores para uma experiência personalizada.

⭐️ Experimente antes de comprar! ⭐️
Antes de comprar este mostrador, confira primeiro a versão gratuita para ter certeza de que é o que você quer e que gosta de como fica no seu relógio: https://apps.garmin.com/apps/65210f56-42d1-4818-a408-889944d80758

⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Este mostrador é totalmente otimizado para MIPS, mas as cores parecerão diferentes como na maioria das imagens, pois estas são tiradas em um dispositivo Amoled. Este mostrador tem muitos temas otimizados para MIPS e suporta gradientes MIPS. Você pode encontrar todos os temas integrados aqui: https://watchface.io/datadash/themes#MIPS

⭐️ Configuração ⭐️
Nota: Existem dois lugares diferentes com configurações:
* App Garmin IQ (no seu telefone vá para seu dispositivo, mostradores, configurações): Configure campos de dados, cores e configurações gerais.
* Configurações no Relógio: Acesse opções mais detalhadas como formato de hora, metas e unidades diretamente no relógio mantendo pressionado o botão menu, selecionando mostrador e tocando em personalizar.

⭐️ Campos de dados ⭐️
Todos os campos de dados que a Garmin atualmente fornece para desenvolvedores terceiros estão disponíveis, mais extras como suporte para Monitor de Glicose (CGM), Open Weather Map (OWM) e suporte para outros apps terceiros.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields . Alguns campos de dados (como prontidão para treino ou HRV) atualmente não estão disponíveis para desenvolvedores terceiros, veja aqui: https://watchface.io/faq/can-you-add-training-readiness
