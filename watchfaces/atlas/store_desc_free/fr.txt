Atlas - Le cadran Garmin ultime riche en données avec 100+ métriques personnalisables, support CGM, prévisions météo et graphiques complets - entièrement optimisé pour les écrans standard et MIPS.

⭐️ Gratuit vs Pro ⭐️
Ce cadran est disponible gratuitement avec toutes les fonctions de base. Les fonctionnalités premium (marquées avec [*]) nécessitent une clé Pro pour débloquer. Vous pouvez acheter une clé Pro ici : https://ko-fi.com/s/d2c3362e83 Ou achetez la version PRO ici : https://apps.garmin.com/apps/490be02d-97dd-495f-95c2-74e8a1b93990

⭐️ Appareils MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Ce cadran est entièrement optimisé pour MIPS, parfait pour le mode de vie nomade numérique, mais les couleurs auront l'air différentes comme dans la plupart des images, car celles-ci sont prises sur un appareil Amoled. Ce cadran a de nombreux thèmes optimisés pour MIPS et supporte les dégradés MIPS. Vous pouvez trouver tous les thèmes intégrés ici : https://watchface.io/atlas/themes#MIPS

⭐️ Configuration ⭐️
Note : Il y a deux endroits différents avec les paramètres :
* App Garmin IQ (sur votre téléphone allez à votre appareil, cadrans, paramètres) : Configurez les champs de données, couleurs et paramètres généraux.
* Paramètres sur la montre : Accédez à des options plus détaillées comme le format de l'heure, les objectifs et les unités directement sur la montre en maintenant le bouton menu, sélectionnant le cadran et tapotant personnaliser.

⭐️ Champs de données ⭐️
Tous les champs de données que Garmin fournit actuellement aux développeurs tiers sont disponibles, plus des extras comme le support du Moniteur de Glucose (CGM), Open Weather Map (OWM) et le support pour d'autres apps tierces.
Tous les champs de données disponibles sont listés ici : https://watchface.io/docs/datafields . Certains champs de données (comme la préparation à l'entraînement ou HRV) ne sont actuellement pas disponibles pour les développeurs tiers, voir ici : https://watchface.io/faq/can-you-add-training-readiness

⭐️ Puis-je changer le texte Fenix/Enduro, etc. ? ⭐️
Oui, vous pouvez ! Voir ici : https://watchface.io/faq/how-to-change-the-fenix-text
En fait, tout sur ce cadran peut être ajusté à votre goût.
