Atlas - O mostrador Garmin definitivo rico em dados com 100+ métricas personalizáveis, suporte CGM, previsões meteorológicas e gráficos abrangentes - totalmente otimizado para ecrãs padrão e MIPS.

⭐️ Gratuito vs Pro ⭐️
Este mostrador está disponível sem custo com todas as funções básicas. As funcionalidades premium (marcadas com [*]) requerem uma chave Pro para desbloquear. Pode comprar uma chave Pro aqui: https://ko-fi.com/s/d2c3362e83 Ou compre a versão PRO aqui: https://apps.garmin.com/apps/490be02d-97dd-495f-95c2-74e8a1b93990

⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Este mostrador está totalmente otimizado para MIPS, perfeito para o estilo de vida nómada digital, mas as cores parecerão diferentes como na maioria das imagens, pois estas são tiradas num dispositivo Amoled. Este mostrador tem muitos temas otimizados para MIPS e suporta gradientes MIPS. Pode encontrar todos os temas integrados aqui: https://watchface.io/atlas/themes#MIPS

⭐️ Configuração ⭐️
Nota: Existem dois locais diferentes com definições:
* App Garmin IQ (no seu telefone vá ao seu dispositivo, mostradores, definições): Configure campos de dados, cores e definições gerais.
* Definições no relógio: Aceda a opções mais detalhadas como formato da hora, objetivos e unidades diretamente no relógio mantendo o botão menu, selecionando mostrador e tocando personalizar.

⭐️ Campos de dados ⭐️
Todos os campos de dados que a Garmin atualmente fornece a programadores terceiros estão disponíveis, mais extras como suporte do Monitor de Glucose (CGM), Open Weather Map (OWM) e suporte para outras apps terceiras.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields . Alguns campos de dados (como prontidão para treino ou HRV) atualmente não estão disponíveis para programadores terceiros, veja aqui: https://watchface.io/faq/can-you-add-training-readiness

⭐️ Posso alterar o texto Fenix/Enduro, etc.? ⭐️
Sim, pode! Veja aqui: https://watchface.io/faq/how-to-change-the-fenix-text
Na verdade, tudo neste mostrador pode ser ajustado ao seu gosto.
